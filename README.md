# Quiz Game in C++

My first C++ project. A simple quiz game that asks users questions, checks their answers, and calculates the final score.

## Features
- Multiple quiz questions
- Instant answer checking
- Final score calculation
- User-friendly console interface

## Concepts Used
- Arrays
- Loops
- If-Else Statements
- Strings
- User Input

## How to Run

Compile:
```bash
g++ quiz_game.cpp -o quiz
```

Run:
```bash
./quiz
```

## Sample Questions
- What is the capital of India?
- How many days are there in a week?
- Which planet is known as the Red Planet?
